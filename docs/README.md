## 📝 文档维护规范

### 文档命名
- 使用英文命名（易于搜索）
- 使用大写字母开头
- 多个单词用下划线连接
- 使用 `.md` 扩展名

### 文档结构
1. 标题和简介
2. 目录（可选）
3. 正文内容
4. 代码示例
5. 常见问题（可选）

### 更新规则
- 功能变更时同步更新文档
- 重大更新记录在文档开头
- 标注新增内容（🆕）
- 归档过时文档（移至 archive/）
